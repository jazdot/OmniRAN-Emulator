# OmniRAN Emulator

----
## Description

OmniRAN Emulator is a unified 5G network emulation solution that simulates real-world UE and gNB behavior. It streamlines 5G testing by delivering an affordable, scalable, and highly accurate virtual RAN environment for next-generation telecommunication validation.

*Note: OmniRAN Emulator borrows libraries and data structures from the [free5gc project](https://github.com/free5gc/free5gc) and builds upon previous open-source solutions.*
